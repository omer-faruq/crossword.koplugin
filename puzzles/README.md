# Local Puzzles Folder

Drop your `.puz` (Across Lite) or `.ipuz` files here and they will appear in
the **Crossword → Library** menu inside KOReader.

## Where to find free puzzles

- [Crosshare](https://crosshare.org) — free community-made crosswords (use the plugin's *Get from Crosshare* menu or download `.puz` manually).
- Washington Post Daily Crossword, LA Times Daily, USA Today, Newsday, The New Yorker, Universal — all publish free daily `.puz` files. Check the per-site licensing.
- [Lil Crossword](https://www.lil-crossword.com), [XWord Info](https://www.xwordinfo.com) free sections, Puzzmo archive, etc.

Licenses vary. Respect the terms of each puzzle's publisher.
